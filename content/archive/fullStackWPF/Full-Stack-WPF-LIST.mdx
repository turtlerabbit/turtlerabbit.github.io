|#01| Domain Introduction and Entity Framework Setup | 프로젝트와 도메인을 소개하고 도메인 계층 구축을 시작하고 애플리케이션을 위한 Entity Framework를 설정한다.|
|#02| Entity Framework CRUD | SimpleTraderDbContext를 약간 리팩터링하고 SimpleTraderDbContext를 인스턴스화하기위한 팩토리 클래스를 만든다. 그런 다음 파트 FULL STACK WFP 01의 Entity Framework 설정을 사용하여 애플리케이션에 대한 Data Service classes 빌드를 시작한다.|
|#03| WPF in .NET and MVVM Navigation | Simple Trader front-end를 위한 WPF project를 설정한다. MVVM 방식으로 프로젝트를 설정하고 매우 간단한 navigation control를 만들고 뷰 모델 간 탐색을 설정한다.|
|#04| API Service Calls and Async ViewModel Loading | 주요 주가 지수에 대한 정보를 로드하는 서비스를 만들어 service layer를 확장한다. 이 서비스는 HttpClient를 사용하여 FinancialModelingPrep API에서 JSON 데이터를 검색하고 Json.NET을 사용하여 JSON을 개체로 역직렬화한다. 그런 다음, I use the service in a ViewModel to load major stock index information.|
|#05| Styling the Navigation Bar | WPF styling으로 navigation bar를 향상시킨다. The navigation bar styles은 프로젝트를 체계적으로 유지하기 위해 자체 resource dictionary에 격리된다. The navigation bar resource dictionary consists of a style for the navigation bar radio buttons. 라디오 버튼 스타일은 사용자 정의 컨트롤 템플릿을 정의하고 다양한 애니메이션과 트리거를 사용한다.|
|#06| Creating a Card Control | MajorIndex 데이터에 대한 card control과 MajorIndexListingViewModel에 대한 user control을 만든다. 그런 다음 HomeView에서 MajorIndex 데이터를 볼 수 있도록 MajorIndexListingViewModel에 대한 컨트롤을 HomeView에 추가한다. 마지막으로 시작시 HomeView를 자동으로 표시하도록 응용 프로그램을 설정했다.|
