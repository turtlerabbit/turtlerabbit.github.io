#
WPF styling으로 navigation bar를 향상시킨다.  
The navigation bar styles은 프로젝트를 체계적으로 유지하기 위해 자체 resource dictionary에 격리된다.  
The navigation bar resource dictionary consists of a style for the navigation bar radio buttons.   
라디오 버튼 스타일은 사용자 정의 컨트롤 템플릿을 정의하고 다양한 애니메이션과 트리거를 사용한다.
#
### 1. The navigation bar styles are isolated in their own resource dictionary
* SimpleTrader.Domain / Styles / NavigationBar.xaml (Resource Dictionary)
    ```js
    <Style x:Key="NavButton" TargetType="RadioButton">
        <Setter Property="Template">
            <Setter.Value>
                <ControlTemplate TargetType="RadioButton">
                    <Grid>
                        <TextBlock Text="{TemplateBinding Content}"
                                   HorizontalAlignment=
                                        "{TemplateBinding HorizontalContentAlignment}" />
                    </Grid>                    
            </Setter.Value>
        </Setter>    
    </Style>
    ```
* SimpleTrader.Domain / App.xaml
    ```js
    ....

    <ResourceDictionary Source="/Styles/NavigationBar.xaml" />

    ....
    ```
* SimpleTrader.Domain / Controls / NavigationBar.xaml
    ```js
    <Grid Grid.Row="1" RenderOptions.EdgeMode="Aliased"
              Background="{StaticResource BrushPrimary2}">
            
            ....

            <Grid.Resources>
                <Style TargetType="RadioButton" BasedOn="{StaticResource NavButton}" />
            </Grid.Resources>
            
            ....
        </Grid>
    ```
* SimpleTrader.Domain / Styles / NavigationBar.xaml (Resource Dictionary)
    ```js
    <Style x:Key="NavButton" TargetType="RadioButton">
        
        ....

        <Setter Property="Foreground" Value="White" />
        <Setter Property="Padding" Value="10 5" />
        <Setter Property="FontSize" Value="18" />

        ....

    </Style>
    ```
* SimpleTrader.Domain / Styles / NavigationBar.xaml (Resource Dictionary)
    ```js
    // 마우스 이벤트 트리거

    <Style x:Key="NavButton" TargetType="RadioButton">        

        ....

        <Style.Triggers>
            <EventTrigger RoutedEvent="MouseEnter">
                <BeginStoryboard>
                    <Storyboard>
                        <ColorAnimation To="{StaticResource ColorPrimary1}"
                                        Duration="0:0:0.1"
                                        Storyboard.TargetProperty="Background.Color" />
                    </Storyboard>
                </BeginStoryboard>
            </EventTrigger>
            <EventTrigger RoutedEvent="MouseLeave">
                <BeginStoryboard>
                    <Storyboard>
                        <ColorAnimation To="{StaticResource ColorPrimary2}"
                                        Duration="0:0:0.1"
                                        Storyboard.TargetProperty="Background.Color" />
                    </Storyboard>
                </BeginStoryboard>
            </EventTrigger>
        </Style.Triggers>
    </Style>
    ```
* SimpleTrader.Domain / Styles / NavigationBar.xaml (Resource Dictionary)
    ```js
    <Style x:Key="NavButton" TargetType="RadioButton">        

        ....

        <Setter Property="Template">

            ....

                    <!-- 라디오 버튼 클릭 시 트리거-->
                    <ControlTemplate.Triggers>
                        <Trigger Property="IsChecked" Value="True">
                            <Setter Property="Background" 
                                    Value="{StaticResource BrushPrimary1}"
                                    TargetName="gridMain" />
                        </Trigger>
                    </ControlTemplate.Triggers>
                </ControlTemplate>
            </Setter.Value>
        </Setter>

    ....

    </Style>
    ```
* SimpleTrader.Domain / Styles / NavigationBar.xaml (Resource Dictionary)
    ```js
    <Style x:Key="NavButton" TargetType="RadioButton">
        <Setter Property="Foreground" Value="White" />
        <Setter Property="Padding" Value="10 5" />
        <Setter Property="FontSize" Value="18" />
        <Setter Property="Background" Value="{StaticResource BrushPrimary2}" />
        <Setter Property="Template">
            <Setter.Value>
                <ControlTemplate TargetType="RadioButton">
                    <Grid x:Name="gridMain" Background="{TemplateBinding Background}">
                        <TextBlock Text="{TemplateBinding Content}"
                                   Padding="{TemplateBinding Padding}"
                                   HorizontalAlignment=
                                        "{TemplateBinding HorizontalContentAlignment}" />
                    </Grid>
                    <!-- 라디오 버튼 클릭 시 트리거-->
                    <ControlTemplate.Triggers>
                        <Trigger Property="IsChecked" Value="True">
                            <Setter Property="Background" 
                                    Value="{StaticResource BrushPrimary1}"
                                    TargetName="gridMain" />
                        </Trigger>
                    </ControlTemplate.Triggers>
                </ControlTemplate>
            </Setter.Value>
        </Setter>
        <Style.Triggers>
            <EventTrigger RoutedEvent="MouseEnter">
                <BeginStoryboard>
                    <Storyboard>
                        <ColorAnimation To="{StaticResource ColorPrimary1}"
                                        Duration="0:0:0.1"
                                        Storyboard.TargetProperty="Background.Color" />
                    </Storyboard>
                </BeginStoryboard>
            </EventTrigger>
            <EventTrigger RoutedEvent="MouseLeave">
                <BeginStoryboard>
                    <Storyboard>
                        <ColorAnimation To="{StaticResource ColorPrimary2}"
                                        Duration="0:0:0.1"
                                        Storyboard.TargetProperty="Background.Color" />
                    </Storyboard>
                </BeginStoryboard>
            </EventTrigger>
        </Style.Triggers>
    </Style>
    ```
